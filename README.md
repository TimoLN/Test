# Version - 1.21.002
## Neuerungen
### Import von Ansprechpartnern
Die Ansprechpartner werden nun täglich auf ABAS importiert. Die Übergabe erfolgt nur bei Neuanlage von Ansprechpartnern innerhalb ABAS.
### TERMINPLANUNG: Neue Sonderzeit: Vertriebsunterstützung
Die Sonderzeit "Vertriebsunterstützung" soll ausschließlich für Techniker benutzt werden, welche den Vertrieb bei Vertriebsgespräche unterstützen.

## Fehlerbehebungen
### #2 - Neuanlage von Benutzern nicht möglich
Im Arbeitsbereich Benutzerverwaltung war das Anlegen von neuen Benutzer nicht möglich, beim Speichervorgang ist ein Fehler aufgetreten.
